Configurable to display an area or line chart.

Markup

API

yDomainMax Put a ceiling on y-axis domain. Default is 100. xFormat Set this value to change the formatting of the the x-axis labels.
  • OneDayHours
    • Expectation is we will be given a range of data from 0-23 and we will map that to a 12 hour clock with am/pm
    • e.g. 0 = 12am, 1 = 1am, 13 = 1pm, 23 = 11pm
numberOfTicks Number of ui intersects in the domain. type If "line", ui will display a line chart. If "area", ui will display an area chart. enableEffects If true, on hover, ui will display continuous values of all lines to the user.
