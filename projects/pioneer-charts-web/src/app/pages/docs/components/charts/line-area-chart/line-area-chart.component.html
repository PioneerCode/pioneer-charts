Configurable to display an area or line chart.

API

domainMax Highest value in domain of data. numberOfTicks Number of ui intersects in the domain. isArea If false, ui will display a line chart. If true, ui will display an area chart. enableEffects If true, on hover, ui will display continues values of all lines to the user.
