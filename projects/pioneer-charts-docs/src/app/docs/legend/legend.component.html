Legend component for displaying chart legends and binding to click events of individual legend items.

Markup

API

Configuration
Property Description Type Default
heading If provided, will display a Heading for the legend. string null
items Collection of legend items to display. PcacLegendConfigItem[] []

Events

Binding Description Payload
(sliceClicked)="onClicked($event)" Emitted when a legend item is clicked. Will include the updated legend item data. PcacLegendConfigItem[]
