Battle Tested Code

Pioneer Charts is built on top of Angular and backed by D3. Of which produces performant and consistent rendering across modern browser.

Charts made simple

Pioneer Charts allows you to create beautiful, consistent and responsive charts with minimal effort.

Standing on the shoulders of giants

Pioneer Charts utilizes Angular as its delivery method. Meaning, if you are familiar with Angular then you are familiar with how the charts inside of Pioneer Charts are added to your project!

Charts are built with the popular data-driven document manipulation library, D3. This allows us to provide rich data-visualizations that are powerful, web-standard compliant, and easy to maintain.
